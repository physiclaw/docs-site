本指南帮助你启动 PhysiClaw 服务器并连接到智能体。你需要已组装好的[硬件](/zh/hardware/bill-of-materials/)、
放入支架的手机，以及 Python 3.11+。

## 安装服务器

<Steps>

1. **安装 Python 依赖。**

   ```sh
   pip install pyserial opencv-python mcp
   ```

2. **克隆仓库。**

   ```sh
   git clone https://github.com/physiclaw/PhysiClaw.git
   cd PhysiClaw
   ```

3. **确认目录结构。**

   <FileTree>
   - physiclaw/
     - physiclaw_server.py   MCP 服务器入口
     - hand.py               串口 G-code 控制
     - eyes.py               USB 摄像头采集
     - config.py             配置常量
   </FileTree>

</Steps>

## 连接硬件

<Steps>

1. 将 **GRBL 控制板**、**顶部摄像头**和**侧向摄像头**接入 USB 集线器，再把集线器接到电脑。

2. 找到串口和摄像头序号，在 `config.py` 中填写：

   <Tabs syncKey="os">
     <TabItem label="macOS">
       ```sh
       ls /dev/tty.usb*        # 例如 /dev/tty.usbserial-1420
       ```
     </TabItem>
     <TabItem label="Linux">
       ```sh
       ls /dev/ttyUSB* /dev/ttyACM*
       ```
     </TabItem>
   </Tabs>

3. 启动服务器，验证它能连上控制板和两个摄像头：

   ```sh
   python -m physiclaw.physiclaw_server --check
   ```

</Steps>

:::caution[权限]
在 Linux 上，将用户加入 `dialout` 组（`sudo usermod -aG dialout $USER`）并重新登录，
这样服务器无需 `sudo` 即可打开串口。
:::

## 在智能体中注册

把 PhysiClaw 添加到你的 MCP 客户端。以 Claude Desktop 为例，编辑 `claude_desktop_config.json`：

```json
{
  "mcpServers": {
    "physiclaw": {
      "command": "python",
      "args": ["-m", "physiclaw.physiclaw_server"]
    }
  }
}
```

重启客户端。PhysiClaw 的工具——`screenshot_top`、`move`、`tap`、`swipe`、`park`——
现在应已可用。继续阅读[快速上手](/zh/start/quickstart/)。
