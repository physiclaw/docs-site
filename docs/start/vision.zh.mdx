个人助手的时代，已经到来。

很快，一个助手会参与到你生活的方方面面。它会帮你点晚餐、买菜，帮你预约看病、清空收件箱。它会规划你的行程、安排你的假期、把日程理得井井有条。你睡着的时候，它也在干活。它会比世上任何人都更懂你——你喜欢什么、每天做些什么、害怕什么。始终在线，始终在你身边，每时每刻，只为你一个人。

但正是让它如此强大的东西，也让它如此危险。要帮你打理整个人生，它就必须看见你的整个人生——你在线上线下做的每一件事。日复一日，它越来越清楚你是谁，直到它掌握的关于你的东西，比你自己能写下来的还要多。

于是，一切都系于一个问题：这一切，归谁所有？

不能是大科技公司。如果他们拥有你的助手，他们就拥有了你。你的数据不只是有用——它是权力。谁握着它，谁就能监视你、塑造你、左右你的选择。一旦你想要的挡了他们想要的，他们随时能反过来对付你。把这样的权力交给少数几家公司，你的帮手就成了你的看守——它威胁你的自由，也威胁我们共同拥有的自由民主社会。

所以，我们把一切建立在一条永不松动的原则之上：你的人生属于你。你的数据，留在你自己的电脑上。命令，由你下达。助手做的一切、学到的一切，都归你所有。没有公司能看到它，没有政府能拿走它。

PhysiClaw 就是我们朝这个方向迈出的第一步。它还很早期——粗糙、未完成、远谈不上完美。但它能用。今天，它已经在为你卸下实实在在的担子，替你处理那些曾经吞掉你大把时间的、手机上的琐碎小事。而它证明了那个更难的论点：另一条路，是走得通的。

它不能成为少数人的特权。所以我们把它开源，也把它送出去——硬件和软件，都送。任何人都能拥有一台。唯一的代价，就是把它造出来的那些零件——便宜到几乎人人都付得起。每个人，都有一个属于自己的助手。不被任何垄断者所掌控。没有人被挡在门外，也没有人被迫攥着另一个人的数据不放。

这，就是我们正在建造的未来：一个只听命于你、不听命于任何其他人的个人助手——由你拥有，人人都触手可及，任何公司或政府都无法染指。

这，就是我们守住自己人生的方式。这，就是我们保持自由的方式。
