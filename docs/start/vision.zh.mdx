个人助手的时代，已经到来。

很快，一个助手就会走进你生活的方方面面。它帮你点晚餐、买菜，帮你挂号看病、清空收件箱；替你规划行程、安排假期，把日程理得井井有条。你熟睡时，它仍在为你忙碌。它会比世上任何人都更懂你——懂你喜欢什么、每天在做什么、又在害怕什么。永远在线，始终在你身边，每时每刻，只为你一个人。

但让它如此强大的，恰恰也让它如此危险。要替你打理整个人生，它就必须看遍你的整个人生——你在线上线下做过的每一件事。日复一日，它越来越清楚你是谁，直到它对你的了解，比你自己写得下来的还要多。

于是，所有问题最终都归结为一个：这一切，究竟归谁所有？

不能是大科技公司。他们若拥有你的助手，就等于拥有了你。你的数据不只是有用——它本身就是权力。谁握住它，谁就能监视你、塑造你、左右你的每一个选择；一旦你想要的，挡了他们想要的，他们随时可以掉转头来对付你。把这样的权力交到少数几家公司手里，你的帮手就成了你的看守——它威胁的，是你的自由，也是我们共同拥有的那个自由而民主的社会。

所以，我们把一切都立在一条永不动摇的原则之上：你的人生，属于你自己。你的数据，留在你自己的电脑里；命令，由你来下；助手所做的一切、所学到的一切，都归你所有。任何公司都看不到它，任何政府都拿不走它。

PhysiClaw，就是我们朝这个方向迈出的第一步。它还只是个雏形——粗糙、未完成、远谈不上完美。但它能用。今天，它已经在为你卸下实实在在的担子，替你打理手机上那些琐碎小事——那些过去总在悄悄吞掉你大把时间的事。而更要紧的是，它证明了一件原本最难证明的事：另一条路，走得通。

它不该成为少数人的特权。所以我们把它开源，也把它送出去——硬件和软件，全都送。任何人都能拥有一台，唯一的成本，不过是把它组装起来的那些零件——便宜到几乎人人都付得起。每个人，都拥有一个真正属于自己的助手，不受任何垄断者掌控。没有人被挡在门外，也没有人能霸占别人的数据。

这，就是我们正在建造的未来：一个只听命于你、绝不听命于他人的个人助手——由你独自拥有，又向每个人敞开，任何公司或政府都无法染指。

这，就是我们守住自己人生的方式；这，就是我们守住自由的方式。
