个人助手的时代，已经到来。

很快，一个助手会参与到你生活的方方面面。它会帮你点晚餐、买菜，帮你预约看病、清空收件箱。它会安排你的行程、张罗你的假期、把日程理得井井有条。你睡着的时候，它也在干活。它会比任何人都更懂你——你喜欢什么、每天做些什么、在为什么发愁。始终在线，始终在你身边，每时每刻，只为你一个人。

要造出这样的助手，我们需要正确的根基。

而这里有一个真实的危险。一个要帮你打理整个人生的助手，必须看见你的整个人生——你在线上线下做的每一件事。日复一日，它越来越清楚你是谁。它掌握的关于你的东西，会比你自己能写下来的还要多。

那么，请只问一个问题：这一切，归谁所有？

不能是大科技公司。如果他们拥有你的助手，他们就拥有了你。你的数据不只是好用——它是权力。谁握着它，谁就能监视你、塑造你、左右你的选择。而一旦你想要的挡了他们想要的，他们随时会反过来对付你。把这样的权力交到少数几家公司手里，它就不再是帮手，而成了威胁——威胁你的自由，也威胁我们共同拥有的自由与民主社会。

当事关你的整个人生时，唯一能让你完全信任的人，只有你自己。

所以，我们把这一切建立在一条简单的原则之上。你说了算。你的数据，留在你自己的电脑上。命令，由你下达。助手创造的一切，都归你所有。你的人生，依旧属于你。

PhysiClaw 是第一步。

它还很早期——粗糙、未完成、远谈不上完美。但它能用。就在此刻，它已经在为你卸下实实在在的担子，替你做那些曾经吃掉你一整天的、手机上的琐碎小事。而它证明了一件更大的事：另一条路，是走得通的。

而它不能只属于少数幸运儿。所以我们把它开源，也把它送出去——机器和软件，都送。任何人都能拥有一台。唯一的花费，就是把它拼起来的那些零件——一个几乎人人都付得起的价格。

每个人，都有一个自己的助手。不被任何巨头所拥有。没有人因为被挡在门外而被落下。也没有谁，被允许攥着另一个人的数据不放。

我们正在建造这样一个未来：你的个人助手属于你，且只属于你一个人——由你拥有、由你驱动，任何公司、任何政府都无法染指。

这，就是我们保持自由的方式。
