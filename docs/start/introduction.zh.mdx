**PhysiClaw** 是一台小型桌面机器人，为 AI 智能体提供一个*物理*身体来操作手机。
它通过摄像头观察屏幕，用三轴机械臂上的电容触控笔触碰玻璃。手机上无需安装任何 App、
无需对接任何 API、无需连接任何账号——对手机来说，PhysiClaw 与人类手指别无二致。

## 问题所在

号称"会用你手机"的软件智能体通常依赖以下三者之一，而每一种都是一堵墙：

<CardGrid>
  <Card title="API 与 OAuth" icon="setting">
    每个服务都需要各自的对接、密钥和授权页面。换个 App，又是一堵墙。
  </Card>
  <Card title="无障碍接口" icon="puzzle">
    读屏与自动化框架可被屏蔽、可被检测，并且会在界面改版时失效。
  </Card>
  <Card title="越狱破解" icon="warning">
    为注入触摸而 root 设备既脆弱又不安全，对大多数人而言也不可行。
  </Card>
</CardGrid>

## PhysiClaw 的方式

与其搭建软件桥梁，不如给智能体一个**实体存在**。这个循环刻意保持简单：

1. **顶部摄像头**拍下屏幕。
2. **智能体**根据图像决定下一步动作。
3. **机械臂**移动触控笔，**侧向摄像头**确认笔尖对准目标。
4. 触控笔**触碰**，随后收回——循环重复。

由于触碰手机的只是一个笔尖，*任意* App 都能用——iOS 或 Android、你的银行、外卖、聊天——
零逐 App 配置。

:::tip[适合谁？]
想让智能体在真实世界中行动的开发者：自动化爱好者、机器人学习者，
以及厌倦了为"一根手指就能完成的事"再写一个对接的所有人。
:::
