PhysiClaw 是一个能操作手机的智能体：它用相机观察屏幕，用触控笔点按，像人一样操作手机。

## PhysiClaw 能做什么？

现有的 AI 智能体主要面向技术场景，最擅长编程、文件处理等工作。但大多数人并不写代码，日常需求集中在手机端：网购、点外卖、买火车票、医院挂号、叫车等高频操作。这些任务与智能体当前擅长的领域几乎没有交集。

PhysiClaw 针对的正是这一类日常事务。它不要求用户掌握任何技术，也不依赖 App 提供接口，而是直接操作手机完成上述操作——凡是用户能在手机上手动完成的任务，PhysiClaw 都能代为执行。

## PhysiClaw 与 OpenClaw

| | PhysiClaw | OpenClaw |
|---|---|---|
| 擅长 | 日常例行事务 | 编程与技术工作 |
| 操作方式 | 物理操作手机 | 命令 / API / 浏览器自动化 |
| 硬件 | 专属硬件 + 手机 | 无 |
| 读取修改电脑文件 | 否 | 是 |
| 安全边界 | 不执行命令 | 拥有系统权限 |
| 被平台检测 | 物理操作难以识别 | 软件操作易被识别 |
| 微信聊天 | 直接和好友聊天 | 需逆向协议或第三方适配 |
