PhysiClaw is an agent that operates a phone: it watches the screen with a camera and taps with a stylus, working the phone the way a person would.

## What PhysiClaw does

Today's AI agents are built mainly for technical work—they're best at coding, file handling, and the like. But most people don't write code; their daily needs live on the phone: shopping, ordering takeout, booking rides, paying bills, making reservations. These tasks barely overlap with what agents are currently good at.

PhysiClaw is built for exactly this kind of everyday task. It asks nothing technical of you and doesn't depend on apps exposing an API—it operates the phone directly. Anything you can do by hand on your phone, PhysiClaw can do for you.

## PhysiClaw vs. OpenClaw

| | PhysiClaw | OpenClaw |
|---|---|---|
| Best at | Everyday routine tasks | Coding and technical work |
| How it operates | Physically operates a phone | Commands / APIs / browser automation |
| Hardware | Dedicated hardware + phone | None |
| Reads/writes computer files | No | Yes |
| Security boundary | Runs no commands | Has system-level access |
| Detectable by platforms | Physical actions are hard to detect | Software actions are easily flagged |
| Chat on WhatsApp | Chats with friends directly | Needs protocol reverse-engineering or a third-party integration |
