## 接下来的路

文档余下的部分，就是这套构建，按顺序来。每一步都干净利落地交接给下一步：

<Steps>

1. **[构建硬件](/zh/hardware/bill-of-materials/)** —— 下单买齐零件，然后把机械臂、触控笔、相机和手机支架组装起来。

2. **[安装软件](/zh/set-up/install/)** —— 一条命令装好 `physiclaw` CLI；`physiclaw doctor` 检查你的机器。

3. **[准备手机](/zh/set-up/prepare-the-phone/)** —— 打开 AssistiveTouch，添加那三个 iOS Shortcuts。五分钟，一次搞定。

4. **[校准](/zh/set-up/calibrate/)** —— 一个向导教机械臂和相机屏幕到底在哪儿，好让 tap 落在 agent 瞄的地方。

5. **[跑你的第一个任务](/zh/run/first-task/)** —— 接上你的 agent，用大白话给它一个目标。看着它看、决策、tap。

</Steps>

:::tip[想先搞懂概念？]
如果你来这儿是想搞懂 PhysiClaw、而不是造一台，那就读[它如何工作](/zh/start/how-it-works/) —— 整套构建服务的，就是那个心智模型。
:::
