给 PhysiClaw 准备一部**专用 iPhone**。它不需要是旗舰或最新机型，一部便宜的二手 iPhone 就完全够用。

:::caution[PhysiClaw 将拥有这部 iPhone 的完全操作权限]
我们已尽力让智能体保护你的隐私、稳定运行、抵御攻击，但仍请你务必做到以下几点：

- **退出 Apple ID 登录**——以免它接触到你的 iCloud 照片、通讯录和「查找」。
- **不要用这部手机的号码绑定你的重要账户**——否则控制这部手机就约等于能重置那些账户。
- **不要把任何账户的密码告诉智能体**。
- **只安装你愿意让 PhysiClaw 使用的 App**。
- **不要安装敏感 App**——不装银行 App、不装支付或钱包 App、不装密码管理器。
- **如果确实需要支付**，可开启**小额免密**，并让该支付账户**只保留少量余额**，把潜在损失控制在可承受范围内。
:::

## 设置微信

PhysiClaw 通过微信和你交流。

### 添加朋友

在 PhysiClaw 使用的手机上登录微信，和你自己使用的手机互加为好友，置顶该聊天，让它始终固定在聊天列表最上方。

### 把微信放到底部栏

把微信图标拖到底部栏——这样无论停在哪一屏，PhysiClaw 都能快速找到并打开它。

:::tip
微信是 PhysiClaw 和你沟通的唯一通道。图标和聊天的位置固定，它才能可靠地查看和回复消息。
:::

## 配置快捷指令


:::tip[连到同一个局域网]
把 iPhone 和运行 PhysiClaw 的电脑接入**同一个 Wi-Fi**——快捷指令要通过局域网和电脑通信。
:::

先获取运行 PhysiClaw 的那台电脑的主机名：

<Tabs syncKey="os">
<TabItem label="macOS">

```bash
scutil --get LocalHostName
```

</TabItem>
<TabItem label="Windows">

```powershell
hostname
```

</TabItem>
<TabItem label="Linux">

```bash
hostname
```

</TabItem>
</Tabs>

得到主机名后，手机桥接页的地址就是 `<hostname>.local:8049`（控制端口 + 1）。也可以用电脑的局域网 IP 访问（例如 `192.168.1.20:8049`），但 IP 由 DHCP 动态分配，电脑重启后可能改变，所以更推荐用 `.local`。

:::note[如果 `.local` 无法解析]
`.local` 依赖 mDNS（macOS 原生支持）。若 iPhone 始终连不上，就改用局域网 IP 连接。
:::

:::tip
下文所有的 `<hostname>` 都要替换成你自己的主机名。
:::

### 屏幕截图

截取当前屏幕并存入「照片」，供「上传最新的截屏」使用。

<Steps>

1. 在**快捷指令** App 里，点 **+** 新建。

2. 添加「**截屏**」。

3. 重命名为 `PhysiClaw Take Screenshot`。

</Steps>

### 上传最新的截屏

当摄像头拍到的屏幕不够清晰、智能体需要更高清画面时使用。

<Steps>

1. 在**快捷指令** App 里，点 **+** 新建。

2. 添加「**获取最新的截屏**」。

3. 添加「**获取 URL 内容**」：
   - URL：`http://<hostname>.local:8049/api/bridge/screenshot`
   - **显示更多** → 方法 **POST**，请求体 **文件**，文件 →「**最新截屏**」变量

4. 重命名为 `PhysiClaw Upload Screenshot`。

</Steps>

### 拷贝到剪贴板

智能体把要输入的文字先传到手机剪贴板再粘贴，比用触控笔逐键点击键盘更快、更可靠。

<Steps>

1. 在**快捷指令** App 里，点 **+** 新建。

2. 添加「**获取 URL 内容**」，URL 填 `http://<hostname>.local:8049/api/bridge/clipboard`，方法选 **GET**。

3. 添加「**拷贝到剪贴板**」，输入设为上一步的「**URL 内容**」。

4. 重命名为 `PhysiClaw Clipboard`。

</Steps>

## 开启辅助触控

辅助触控就是 iOS 屏幕上那个悬浮按钮（俗称「小白点」）——机械臂通过点击它来触发快捷指令。开启后，把三种手势分别映射到三个快捷指令。

<Steps>

1. **设置 → 辅助功能 → 触控 → 辅助触控** → 打开。

2. 在**自定操作**里设置：
   - **单点** → **PhysiClaw Take Screenshot**
   - **轻点两下** → **PhysiClaw Upload Screenshot**
   - **长按** → **PhysiClaw Clipboard**

</Steps>


## 设置锁屏密码

PhysiClaw 能自己解锁手机，但它只会使用这一个固定密码：`111111`。用专用密码，可以避免你的常用密码经由日志或配置文件泄露。请把这部手机的锁屏密码设为 `111111`：

**设置 → 面容 ID 与密码 → 打开密码 → 输入 `111111`**

:::note[关闭自动锁屏]
关闭自动锁定（**设置 → 显示与亮度 → 自动锁定 → 永不**），让手机始终保持解锁。代价是屏幕长亮，老化更快。
:::

## 固定屏幕亮度

摄像头会根据屏幕亮度调整曝光时间，屏幕自动亮度调节可能导致拍摄画面过曝或欠曝。请关闭自动亮度调节，并将屏幕亮度固定在 50% 或更高。

**设置 → 辅助功能 → 显示与文字大小 → 自动亮度调节 → 关闭**

## 允许从其他 App 粘贴

PhysiClaw 通过快捷指令，把文字复制到剪贴板，再粘贴到微信与你沟通，或粘贴到淘宝、京东搜索商品。
如果每次粘贴都弹窗询问，可按如下设置免去提示：

**设置 →〔App 名称〕→ 从其他 App 粘贴 → 允许**
