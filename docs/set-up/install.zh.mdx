---
title: 安装软件
description: 用一条脚本安装 physiclaw CLI，再用 --version 和 doctor 验证。
---

PhysiClaw 的软件部分是一个命令行工具（CLI）：`physiclaw`。安装完成后，确认版本，并运行健康检查。

## 1. 运行安装脚本

<Tabs syncKey="os">
<TabItem label="macOS / Linux">

```bash
curl -fsSL https://physiclaw.ai/install.sh | bash
```

</TabItem>
<TabItem label="Windows">

```powershell
iwr -useb https://physiclaw.ai/install.ps1 | iex
```

</TabItem>
</Tabs>

这会安装 [`uv`](https://docs.astral.sh/uv/)、Python 3.12 和 `physiclaw` CLI。

## 2. 确认安装成功

```bash
physiclaw --version
```

如果终端提示找不到该命令，请关闭当前**终端**窗口，重新打开后再试一次。

## 3. 运行健康检查

```bash
physiclaw doctor
```

`doctor` 会报告当前版本、配置路径、本地视觉模型、检测到的硬件、配置的大模型、安装的技能。

## 后续更新

PhysiClaw 会自动保持最新：每次启动 `physiclaw server` 时都会检查 PyPI，并自动重启到最新版本。想手动更新，或者已经关闭了自动更新时：

```bash
physiclaw update           # 更新到最新版本
physiclaw update --check   # 只检查，不安装
```

关闭自动更新：

```bash
physiclaw config set update.auto false
```
