---
title: 安装软件
description: 用一条脚本安装 physiclaw CLI，再用 --version 和 doctor 验证。
---

PhysiClaw 的软件部分是一个命令行工具（CLI）：`physiclaw`。安装完成后，确认版本，并运行健康检查。

## 1. 运行安装脚本

<Tabs syncKey="os">
<TabItem label="macOS / Linux">

```bash
curl -fsSL https://physiclaw.ai/install.sh | bash
```

</TabItem>
<TabItem label="Windows">

```powershell
iwr -useb https://physiclaw.ai/install.ps1 | iex
```

</TabItem>
</Tabs>

这会安装 [`uv`](https://docs.astral.sh/uv/)、Python 3.12 和 `physiclaw` CLI，随后下载本地视觉模型。若模型下载失败，可用 `physiclaw setup local-vision-model` 重新下载。

## 2. 确认安装成功

```bash
physiclaw --version
```

如果终端提示找不到该命令，请打开一个**新的终端**窗口，让更新后的 `PATH` 生效，再试一次。

## 3. 运行健康检查

```bash
physiclaw doctor
```

`doctor` 会报告当前版本、配置路径、本地视觉模型、检测到的硬件、配置的大模型、安装的技能。

## 更新

PhysiClaw 以 `uv` 工具形式安装。先停止服务器，再用 uv 更新：

```bash
uv tool upgrade physiclaw
```
