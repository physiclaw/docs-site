---
title: 安装软件
description: 用一条脚本安装 physiclaw CLI，再用 --version 和 doctor 验证。
---

PhysiClaw 的软件部分是一个命令行工具（CLI）：`physiclaw`。安装完成后，确认版本，并运行健康检查。

## 1. 运行安装脚本

<Tabs syncKey="os">
<TabItem label="macOS / Linux">

```bash
curl -fsSL https://physiclaw.ai/install.sh | bash
```

</TabItem>
<TabItem label="Windows">

```powershell
irm https://physiclaw.ai/install.ps1 | iex
```

</TabItem>
</Tabs>

这会安装 [`uv`](https://docs.astral.sh/uv/)、Python 3.12 和 `physiclaw` CLI。

## 2. 确认安装成功

```bash
physiclaw --version
```

如果终端提示找不到该命令，请关闭当前**终端**窗口，重新打开后再试一次。

## 3. 运行健康检查

```bash
physiclaw doctor
```

`doctor` 会报告 Python 版本、配置路径、本地视觉模型、检测到的硬件。
