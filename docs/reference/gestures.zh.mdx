---
title: 手势
description: 点按、长按、滑动与双击如何转化为 GRBL G-code。
---

每个手势都是一小段 G-code。Z 轴承载触控笔：`M3 S12` 让笔下探触碰玻璃，`M3 S0` 抬起。
`G4 P<秒>` 停留；`G0`/`G1` 移动。

## 单击

移动到目标，下探 50–100 ms，抬起。

```text title="G-code"
G0 X.. Y..      ; 移动到目标
M3 S12          ; 笔下探
G4 P0.05        ; 停留 50 ms
M3 S0           ; 笔抬起
```

## 长按

与单击相同，但停留时间保持约 800 ms。

```text title="G-code"
G0 X.. Y..
M3 S12
G4 P0.8         ; 保持 800 ms
M3 S0
```

## 滑动

在起点下探，带载移动，在终点抬起。

```text title="G-code"
G0 X.. Y..      ; 移动到起点
M3 S12          ; 笔下探
G1 X.. Y.. F3000 ; 以进给 3000 拖到终点
M3 S0           ; 笔抬起
```

## 双击

两次点按之间间隔很短（两次触碰间 &lt; 300 ms）。

```text title="G-code"
; 点按
M3 S12
G4 P0.05
M3 S0
G4 P0.1         ; 间隔 100 ms
; 点按
M3 S12
G4 P0.05
M3 S0
```

:::tip[调节触摸]
如果点按不触发，略微增大 Z 深度（`S` 值）或停留（`P`）。如果划屏或重复触发，则减小它们。
数值由[标定](/zh/start/quickstart/)保存。
:::

## 汇总

| 手势 | 下探 | 停留 | 移动 | 抬起 |
| ---- | ---- | ---- | ---- | ---- |
| 点按 | `M3 S12` | `P0.05` | — | `M3 S0` |
| 长按 | `M3 S12` | `P0.8` | — | `M3 S0` |
| 滑动 | `M3 S12` | — | `G1 F3000` | `M3 S0` |
| 双击 | ×2 | `P0.05` | — | ×2 |
