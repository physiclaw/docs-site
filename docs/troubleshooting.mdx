## PhysiClaw exits on its own after a while

PhysiClaw runs fine, then quits after some time. This usually happens because, with no user activity for a while, the operating system turns the camera off — PhysiClaw can no longer read a photo of the phone screen, so it exits.

:::tip[Fix]
Make sure the computer running PhysiClaw won't lock the screen or go to sleep.
:::
