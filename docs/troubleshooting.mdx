## PhysiClaw exits on its own after a while

PhysiClaw runs fine but quits on its own after a while. This usually happens because the system shuts off the camera after a long stretch of inactivity — without it, PhysiClaw can't capture the phone screen, so it exits.

:::tip[Fix]
Make sure the computer running PhysiClaw won't lock the screen or go to sleep.
:::

## Camera preview works, but calibration keeps failing

The camera looks fine when you check it in a video or camera app, yet calibration fails as soon as you run it. This is usually a lighting issue: the room is too dark, a light is shining directly onto the phone screen and causing glare, or the screen brightness is too low.

:::tip[Solution]
Make sure the room is evenly lit, avoid glare on the screen, and set the phone's screen brightness to a suitable level.
:::
