[采购页](/zh/hardware/sourcing/) 把整套搭建分成了三类。本页讲的全是 **C 类**：没有供应商现货、要你自己生产的定制零件。

「自己生产」**不等于**在家打印。PhysiClaw 的定制件都是承力配合件——它们承托运动龙门、定位惰轮、把电磁铁稳稳压在手机正上方——所以尺寸必须做准。这就把桌面级 FDM 打印机排除在外了。你在这一步真正要做的，是生成一个几何文件包，交给跑粉末床熔融工艺的 **3D 打印服务商**。

本页的成果：一份供应商可直接用的 **STEP 文件** zip 包，数量正确，用 **黑色 PA12 尼龙、SLS 或 MJF 工艺** 打印。然后你就可以进入 [组装](/zh/hardware/assembly/) 环节。

:::note[本页讲的是*做出*这些零件，不是把它们装上去]
你在这里要做的是下单并生产这些打印件。每个零件装在哪、怎么装，是 [组装](/zh/hardware/assembly/) 页的内容。这一步要和切割型材、订购通用件**并行**进行，而不是等它们之后再做，因为打印排队是整套搭建中最长的交期之一。
:::

## 你需要什么：9 个设计，11 件实物

一共有 **9 个独立设计**，但 **11 件实物**，因为惰轮支架要打印多份。其余都是单件。下面是完整清单，以及每个零件在搭建中落在哪里，方便你理解它为什么存在：

| 设计 | 数量 | 在搭建中的作用 |
| --- | --- | --- |
| **IdlerMountFront** | 2 | 在龙门上定位前端 GT2 惰轮（12 惰轮 CoreXY 走带的一部分）。 |
| **IdlerMountMotor** | 2 | 在电机/驱动轮一端定位惰轮。 |
| **XY joint (left)** | 1 | 在 MGN9H 滑座上承托运动龙门和触控笔滑架——左侧。 |
| **XY joint (right)** | 1 | 同上，右侧。两个 joint 是镜像件，不能互换。 |
| **Belt clamp** | 1 | 在滑架处夹紧并张紧 GT2 同步带。 |
| **Solenoid mount** | 1 | 固定按压 Z（即「点触」）的 12V 推拉式电磁铁。 |
| **Phone bed** | 1 | 把手机固定在相机下方的支架里。 |
| **Control-board holder** | 1 | 把 MKS DLC32 V2.1 控制板装到机架上。 |
| **Tube holder** | 1 | 沿龙门走 PTFE/FEP 管。 |

加起来就是 `2 + 2 + 1×7 = 11` 件。唯一需要订多份的就是惰轮支架，而且它有**两个变体**——前端和电机端——所以每个变体**各订 2 个**，而不是同一个订 4 个。

## 材料和工艺——没有商量余地

这些零件必须用一种特定方式制造。别让工厂给你报个更便宜的方案。

| 规格 | 值 |
| --- | --- |
| 材料 | **黑色 PA12 尼龙** |
| 工艺 | **粉末床熔融——只能 SLS 或 MJF** |
| 密度 | 实心（100%） |
| 层高 | 100 µm |
| 最小壁厚 | 1.0 mm |
| 表面 | 原始/打印态 |

:::caution[FDM 被排除——原因就在这里]
这些零件中有几个带**侧向（横向）孔**和**悬垂**。在 FDM 打印机上，侧孔会随着塑料逐层堆叠而变形，悬垂没有支撑会塌陷——结果就是 MGN9H 滑座装不进去、电磁铁摆不正、皮带夹夹不住。SLS 和 MJF 在**粉末床**里成型零件、无需支撑，所以悬垂和孔在各个方向上都做得准。PA12 尼龙、SLS 或 MJF，没有别的选项。家用 FDM 机器做不了这些零件。
:::

## 几何文件从哪来

权威几何是仓库里的 **build123d Python 零件定义**。你运行一个导出器把它们转成 **STEP 文件**，发给工厂的就是这些 STEP 文件。

:::caution[不要发那些遗留的 FreeCAD 文件]
`hardware/custom/` 里有五个旧的 `.FCStd` 文件——`PhysiClaw_BeltClamp`、`PhysiClaw_Pulley_Front`、`PhysiClaw_Pulley_Motor`、`PhysiClaw_SolenoidMount` 和 `PhysiClaw_XY_Joint`。它们是**遗留文件，并非权威**。不要导出它们，也不要交给打印商。下面这套 build123d → STEP 包才是唯一可信来源。
:::

## 生成 STEP 包

导出器在 `hardware/parts/export_custom.py`。它会为**每个零件写出一个 `.step` 文件**到 `hardware/output/step/`。在仓库根目录运行：

```bash
uv run python -m hardware.parts.export_custom
```

这会导出全部 9 个设计。数量已经固化在导出器里（`IdlerMountFront` qty=2、`IdlerMountMotor` qty=2，其余七个 qty=1），所以文件列表与上表一致。

然后把结果打包给供应商：

<Steps>

1. **检查输出。** 确认 `hardware/output/step/` 里每个设计都有一个 `.step` 文件。

2. **把 STEP 文件打成一个压缩包** 以便上传。

3. **在订单里写清数量，不要只给文件名。** 工厂按你说的打印，所以要逐一写明：IdlerMountFront ×2、IdlerMountMotor ×2、其余各 ×1。

4. **附上上表中的材料/工艺规格**——黑色 PA12、SLS 或 MJF、实心、100 µm、最小壁厚 1.0 mm、原始表面。

5. **把 zip 上传到你的打印服务商** 并下单。

</Steps>

## 成本和交期，实话实说

在中国，这批 3D 打印件约 **¥190**——是约 $145 / 约 ¥1046 零件总预算里的一项。在**美国/欧洲市场会更贵**，因为粉末床打印在那里更贵。

这是一个**要和供应商沟通、跨好几天**的步骤，不是一下子就能完成的。你上传 STEP zip、和工厂确认每个零件的数量，然后等打印排队和交期。正因为这段等待很长，请**尽早**下这个单，并和切割型材、采购通用件**并行**进行——别把它当成等其他东西都到齐之后才做的事。

:::tip[先下打印单]
三类采购里，打印件和切割型材的交期最长。先把这个 STEP 单发出去，再去张罗紧固件和耗材，让最慢的那部分先进排队，你同时处理其余的。
:::
