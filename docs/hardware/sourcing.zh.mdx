PhysiClaw 的 X/Y 运动是一台手工搭建的 **CoreXY 龙门**，要按原始零件来采购 —— 没有现成的机械臂。
零件成本大约 **$145 美元**（在中国约 ¥1046；美元/欧元市场更高），分摊在一长串清单上，
所以下单的*流程*比单件砍价更重要。

完整的逐项 **BOM**、买 / 切 / 打印三类采购方式、实际花销，以及让慢件并行赶制的下单顺序，
都在**采购指南**里。9 个定制件以 STEP 文件交给 3D 打印服务。

:::caution[用 PA12 + SLS/MJF 打印，不要用 FDM]
这些定制件是承重的配合件。请用**黑色 PA12 尼龙、粉末床熔融（SLS 或 MJF）**打印。
FDM 不行 —— 侧孔会变形、悬垂会塌。
:::

零件到手后，[组装](/zh/hardware/assembly/) 把它们组装起来；然后刷 [固件](/zh/hardware/firmware/)。
