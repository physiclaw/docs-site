PhysiClaw 硬件优先采用标准零件——易于采购、成本低、可靠；定制零件采用 3D 打印。

整套零件成本约需 **¥1,000**（不含运费）。下单前请通读装配手册，充分了解硬件结构；采购时逐项核对物料清单，确保无遗漏、数量准确——任何缺漏都需补单，物流将耗费更多时间。供应商均为淘宝店铺，需登录淘宝账号查看商品详情。

:::caution[3D 打印工艺]
PhysiClaw 定制零件请用黑色 **[PA12 尼龙材料](https://en.wikipedia.org/wiki/Nylon_12)**，以粉末床熔融工艺（**[SLS](https://en.wikipedia.org/wiki/Selective_laser_sintering)** 或 **[MJF](https://en.wikipedia.org/wiki/Multi_Jet_Fusion)**）打印。
常见的 **[FDM](https://en.wikipedia.org/wiki/Fused_filament_fabrication)** 工艺会导致孔位失圆、变形，影响装配。
:::
