这一页是完整的零件与下单参考：每个零件 *是什么*、*花多少钱*，以及怎么把它搬上你的书桌——哪些直接现成买、哪些得让供应商替你切割或打印，以及该按什么顺序下单，好让慢件并行赶制。在你下单、切割或打印任何东西之前，先读它。

先把一件事说清楚：**没有现成的整臂**。PhysiClaw 的 X/Y 运动是一套手工搭的 **CoreXY 龙门**，你得按散件去采购——铝型材、导轨、同步带、步进电机、一块控制板——然后自己组装。没有任何东西会以「整臂」的形态预装好寄来；Z 轴是一个你自己接线的 **solenoid**，不是绘图笔。这意味着实打实的采购工作：跟供应商沟通切铝、把文件传给 3D 打印服务、是的，还有一点焊接来接线。给零件到货留出几天时间，别指望一个下午搞定。

## 要花多少钱

按零件预算大约 **$145 美元**（在国内约 **¥1046**，按 ~7.2 CNY/USD 折算）。如果你从这些零件本就所属的、以中国为中心的供应链来采购，总价最低；同样一套搭建在 **美元/欧元市场会更贵**，主要贵在导轨、步进电机和加工过的铝型材上。把这些数字当作以中国为中心的下限，而不是固定价格。

拿下面这份逐项参考价，去核一核你拿到的任何报价（国内价格，¥）：

| 零件 | 约价 |
| --- | --- |
| 相机（4K/8MP 自动对焦 USB） | ¥200 |
| 3D 打印件（9 个设计 / 11 件） | ¥190 |
| MGN9H 导轨 + 滑块 | ¥120 |
| MKS DLC32 V2.1 控制板 | ¥100 |
| 铝型材（切割/攻丝） | ¥60 |
| 2× NEMA 17 步进电机 | ¥50 |
| 12V 2A 电源 | ¥50 |
| 2× TMC2209 驱动 | ¥35 |

紧固件、同步带、惰轮、solenoid 和耗材凑齐剩下的部分。这些没有一样是要特别订制或单价昂贵的——成本是摊在一长串清单上的（螺丝、T 型螺母、垫片、隔套都按 20/50/100 个一包卖，所以便宜的件加起来也不少），这也正是为什么下单的 *流程* 比在某一行上抠一笔折扣更要紧。

## 三类采购方式

完整 BOM 有 250 多件，听上去吓人，但把它归成三种动作就清爽了。每一件都正好落进其中一类：

  <Card title="A——照原样买" icon="rocket">
    现成件，不用改。清单上的绝大多数：MGN9H 导轨 + 滑块、GT2 同步带 / 带轮 / 12 个惰轮、2× NEMA 17 步进电机、solenoid、MKS DLC32 V2.1 控制板、2× TMC2209 驱动、12V 2A 圆口电源、4K/8MP 自动对焦 USB 摄像头，以及所有紧固件和耗材。
  </Card>
  <Card title="B——买来再切" icon="setting">
    铝型材。你不用买整根料再自己手工加工——把精确的切割、攻丝、沉孔规格发给供应商，让他们替你做。
  </Card>
  <Card title="C——送出去打印" icon="document">
    那 9 个自定义设计（11 件）。以 STEP 文件交给 3D 打印服务。它们必须用 PA12 尼龙、以粉末床熔融工艺打印——FDM 不行。
  </Card>

**B** 和 **C** 两类的交期最长，所以先下单。下面的逐项清单按类分组；「照此顺序做」的流程放在最后。

## B 类——切割铝型材

你需要从原始铝型材切出七段。这些都是 **定制切割、攻丝、钻孔** 的——把规格发给供应商，请他们做这些加工。以中国为中心的型材供应商干这个是家常便饭，加价也不多。所有型材都是黑色阳极氧化、欧标截面。

| 零件 | 规格 | 数量 | 加工 |
| --- | --- | --- | --- |
| 侧导轨 | 2040，345 mm | 2 | 沉孔 M6 |
| 端导轨 | 2040，190 mm | 2 | 攻丝 M6 |
| 龙门 X 横梁 | 1020，185 mm | 1 | — |
| 手机座支撑 | 1020，230 mm | 2 | — |

下面是同一份规格的精简版，复制后发给供应商：

- **2× 2040 黑色阳极氧化 EU 型材，345mm**——侧轨，**M6 沉孔**
- **2× 2040 黑色阳极氧化 EU 型材，190mm**——端轨，**攻丝 M6**
- **1× 1020 型材，185mm**——龙门 X 横梁
- **2× 1020 型材，230mm**——手机座支撑

:::caution[第一次就把标注弄对]
沉孔 vs 攻丝的区别、以及精确的长度，都是承重的关键——这些一旦弄错，框架就方不起来，而重切一次又得多等好几天。发消息前，对照手册再核一遍。
:::

## C 类——3D 打印件

这一步多数做软件的人没干过，所以换个角度想：你是把一组几何文件和一份材料规格交给打印服务，他们把成品零件寄给你。这些是 **承力配合件**——它们承托运动龙门、定位惰轮、把 solenoid 稳稳压在手机正上方——所以尺寸必须做准。这就把桌面级 FDM 打印机排除在外了。「自己生产」指的是生成几何文件包，交给跑粉末床熔融工艺的 **3D 打印服务商**。把这份订单写对，就是这步的全部工作。

### 你需要什么：9 个设计，11 件实物

一共有 **9 个独立设计**，但 **11 件实物**，因为惰轮支架要打印多份——它有 **两个变体**（前端和电机端），所以每个变体 **各订 2 个**，而不是同一个订 4 个。其余都是单件（`2 + 2 + 1×7 = 11`）。下面是完整清单，以及每个零件在搭建中落在哪里：

| 设计 | 数量 | 在搭建中的作用 |
| --- | --- | --- |
| **IdlerMountFront** | 2 | 在龙门上定位前端 GT2 惰轮（12 惰轮 CoreXY 走带的一部分）。 |
| **IdlerMountMotor** | 2 | 在电机/驱动轮一端定位惰轮。 |
| **XY joint (left)** | 1 | 在 MGN9H 滑座上承托运动龙门和触控笔滑架——左侧。 |
| **XY joint (right)** | 1 | 同上，右侧。两个 joint 是镜像件，不能互换。 |
| **Belt clamp** | 1 | 在滑架处夹紧并张紧 GT2 同步带。 |
| **Solenoid mount** | 1 | 固定按压 Z（即「点触」）的 12V 推拉式 solenoid。 |
| **Phone bed** | 1 | 把手机固定在相机下方的支架里。 |
| **Control-board holder** | 1 | 把 MKS DLC32 V2.1 控制板装到机架上。 |
| **Tube holder** | 1 | 沿龙门走 PTFE/FEP 管。 |

### 材料和工艺——没有商量余地

照这个精确指定，别让打印厂以「便宜」为由把你劝去 FDM：

| 规格 | 取值 |
| --- | --- |
| 材料 | **黑色 PA12 尼龙** |
| 工艺 | **粉末床熔融——只能 SLS 或 MJF** |
| 密度 | 实心（100%） |
| 层高 | 100 µm |
| 最小壁厚 | 1.0 mm |
| 表面处理 | 毛坯/打印态 |

:::caution[拒绝 FDM——原因就在这里]
这些零件中有几个带 **侧向（横向）孔** 和 **悬垂**。在 FDM 打印机上，侧孔会随着塑料逐层堆叠而变形，悬垂没有支撑会塌陷——结果就是 MGN9H 滑座装不进去、solenoid 摆不正、同步带夹夹不住。SLS 和 MJF 在 **粉末床** 里成型零件、无需支撑，所以悬垂和孔在各个方向上都做得准。PA12 尼龙、SLS 或 MJF，没有别的选项。家用 FDM 机器做不了这些零件。
:::

### 生成并打包 STEP 文件

权威几何是仓库里的 **build123d Python 零件定义**。导出器在 `hardware/parts/export_custom.py`，它会为 **每个零件写出一个 `.step` 文件** 到 `hardware/output/step/`。在仓库根目录运行：

```bash
uv run python -m hardware.parts.export_custom
```

这会导出全部 9 个设计。数量已经固化在导出器里（`IdlerMountFront` qty=2、`IdlerMountMotor` qty=2，其余七个 qty=1），所以文件列表与上表一致。然后把结果打包给供应商：

<Steps>

1. **检查输出。** 确认 `hardware/output/step/` 里每个设计都有一个 `.step` 文件。

2. **把 STEP 文件打成一个压缩包** 以便上传。

3. **在订单里写清数量，不要只给文件名。** 工厂按你说的打印，所以要逐一写明：IdlerMountFront ×2、IdlerMountMotor ×2、其余各 ×1。

4. **附上上表中的材料/工艺规格**——黑色 PA12、SLS 或 MJF、实心、100 µm、最小壁厚 1.0 mm、原始表面。

5. **把 zip 上传到你的打印服务商** 并下单。

</Steps>

:::caution[不要发那些遗留的 FreeCAD 文件]
`hardware/custom/` 里有五个旧的 `.FCStd` 文件——`PhysiClaw_BeltClamp`、`PhysiClaw_Pulley_Front`、`PhysiClaw_Pulley_Motor`、`PhysiClaw_SolenoidMount` 和 `PhysiClaw_XY_Joint`。它们是 **遗留文件，并非权威**。不要导出它们，也不要交给打印商。上面那套 build123d → STEP 包才是唯一可信来源。
:::

## A 类——现成件

B 和 C 一下完，就把 A 类里的一切都订上，让它们并行到货。这些是照原样买的通用件。下面的逐项清单就是完整列表；后面附上几条容易买少的提醒。

### 紧固件与五金（约 250+ 个）

从零搭一台龙门需要大量小五金。预计 **超过 250 个零件**，按 20/50/100 个一包卖：

- **螺丝：** BHCS / FHCS / SHCS，规格 M3、M5、M6，外加 1/4-20。
- **螺母：** 弹片式/滑入式 T 型螺母和方螺母。
- **五金：** 10 个轴肩螺丝、12 个垫片、8 个隔套。
- **支架：** 6 个平角支架、2 个电机支架、1 个角支架。
- **脚垫：** 4 个缓冲脚垫。

**按包买、宁可多订。** 螺丝按 20 / 50 / 100 一包卖——按包买、留点备用，别去数精确个数。哪怕少一颗 M3，也不该把一个跨好几天的搭建给卡住。

:::tip[紧定螺丝松脱是反馈最多的故障]
买一支 **Loctite 243（乐泰 243 螺纹胶）**，涂在驱动带轮的紧定螺丝上。这是一份便宜的保险，防的正是大家搭完之后最常碰到的那个问题。这里的螺纹胶不是可选项。
:::

### 耗材

这些和结构件分开发货，所以很容易忘了订，然后就卡住——和其余件放在同一批订：

| 项目 | 备注 |
| --- | --- |
| 锂基润滑脂 | NLGI 2，用于导轨——导轨出厂带油，要擦掉重新打脂 |
| Loctite 243 | 螺纹胶——见上方提醒 |
| 热缩管 | 用于接线 |
| PTFE/FEP 管 | ~400 mm；**线走在管外**，不是穿过它 |
| 缠绕管（螺旋管） | 理线用 |
| 引线 | 0.3 mm² |

### 手边要备的工具

如果这些你还没有，一并加进订单：

- 2 / 3 / 5 mm 内六角扳手
- 电动起子（建议——螺丝 *真的* 很多）
- 一把 **电烙铁 + 热风枪**——你要接线焊接；这不是免焊的搭建
- 一个用来对框架找方的平面
- 一个手机调音 App，后面给同步带调张力时用

## 真正要紧的是什么

不是每个零件都得做到好。把注意力——和预算——花在真正重要的地方：

## 推荐的下单顺序

交期是最长的那根杆子，所以要把它并行掉：

<Steps>

1. **重读手册、重核上面的完整零件清单。** 切割长度和沉孔/攻丝标注必须第一次就对——切错或漏攻一处，就又是好几天的等待。

2. **先下 3D 打印单（C 类）。** 导出 STEP → 打 zip → 按 PA12 / SLS-或-MJF 规格上传给打印服务。这通常是交期最长的一项。

3. **再下铝型材切割单（B 类）。** 把切割/攻丝/沉孔规格发给供应商。

4. **订现成件（A 类）。** 步进电机、控制板、驱动、导轨、同步带、惰轮、solenoid、相机、电源、紧固件、耗材，以及任何你缺的工具——全放一批里订，好让它们在打印件和型材赶制时同步到货。

</Steps>

这是一个 **要和供应商沟通、跨好几天** 的过程，不是一下子就能完成的。打印件和切割型材的交期最长，所以先把这两单发出去，再去张罗紧固件和耗材。
