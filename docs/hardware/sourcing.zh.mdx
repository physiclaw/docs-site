你已经有了[物料清单](/zh/hardware/bill-of-materials/)——也就是知道了 *有哪些* 零件。这一页是把它们真正搬上你书桌的那座桥：实际要花多少钱、哪些直接现成买、哪些得让供应商替你切割或打印，以及该按什么顺序下单，好让慢件并行赶制。

先把一件事说清楚：**没有现成的整臂**。PhysiClaw 的 X/Y 运动是一套手工搭的 **CoreXY 龙门**，你得按散件去采购——铝型材、导轨、同步带、步进电机、一块控制板——然后自己组装。没有任何东西会以「整臂」的形态预装好寄来；Z 轴是一个你自己接线的 **solenoid**，不是绘图笔。这意味着实打实的采购工作：跟供应商沟通切铝、把文件传给 3D 打印服务、是的，还有一点焊接来接线。给零件到货留出几天时间，别指望一个下午搞定。

## 要花多少钱

按零件预算大约 **$145 美元**（在国内约 **¥1046**，按 ~7.2 CNY/USD 折算）。如果你从这些零件本就所属的、以中国为中心的供应链来采购，总价最低；同样一套搭建在 **美元/欧元市场会更贵**，主要贵在导轨、步进电机和加工过的铝型材上。

拿下面这份逐项参考价，去核一核你拿到的任何报价（国内价格，¥）：

| 零件 | 约价 |
| --- | --- |
| 相机（4K/8MP 自动对焦 USB） | ¥200 |
| 3D 打印件（9 个设计 / 11 件） | ¥190 |
| MGN9H 导轨 + 滑块 | ¥120 |
| MKS DLC32 V2.1 控制板 | ¥100 |
| 铝型材（切割/攻丝） | ¥60 |
| 2× NEMA 17 步进电机 | ¥50 |
| 12V 2A 电源 | ¥50 |
| 2× TMC2209 驱动 | ¥35 |

紧固件、同步带、惰轮、solenoid 和耗材凑齐剩下的部分。这些没有一样是要特别订制或单价昂贵的——成本是摊在一长串清单上的，这也正是为什么下单的 *流程* 比在某一行上抠一笔折扣更要紧。

## 三类采购方式

完整 BOM 有 250 多件，听上去吓人，但把它归成三种动作就清爽了。每一件都正好落进其中一类：

<CardGrid>
  <Card title="A——照原样买" icon="rocket">
    现成件，不用改。清单上的绝大多数：MGN9H 导轨 + 滑块、GT2 同步带 / 带轮 / 12 个惰轮、2× NEMA 17 步进电机、solenoid、MKS DLC32 V2.1 控制板、2× TMC2209 驱动、12V 2A 圆口电源、4K/8MP 自动对焦 USB 摄像头，以及所有紧固件和耗材。
  </Card>
  <Card title="B——买来再切" icon="setting">
    铝型材。你不用买整根料再自己手工加工——把精确的切割、攻丝、沉孔规格发给供应商，让他们替你做。
  </Card>
  <Card title="C——送出去打印" icon="document">
    那 9 个自定义设计（11 件）。以 STEP 文件交给 3D 打印服务。它们必须用 PA12 尼龙、以粉末床熔融工艺打印——FDM 不行。
  </Card>
</CardGrid>

**B** 和 **C** 两类的交期最长，所以先下单。其余的就按下面这套「照此顺序做」的流程走。

## B 类——下单切割铝型材

你需要从原始铝型材切出七段。这些都是 **定制切割、攻丝、钻孔** 的——把规格发给供应商，请他们做这些加工。以中国为中心的型材供应商干这个是家常便饭，加价也不多。

下面这份规格，复制后发给供应商：

- **2× 2040 黑色阳极氧化 EU 型材，345mm**——侧轨，**M6 沉孔**
- **2× 2040 黑色阳极氧化 EU 型材，190mm**——端轨，**攻丝 M6**
- **1× 1020 型材，185mm**——龙门 X 横梁
- **2× 1020 型材，230mm**——手机座支撑

:::caution[第一次就把标注弄对]
沉孔 vs 攻丝的区别、以及精确的长度，都是承重的关键——这些一旦弄错，框架就方不起来，而重切一次又得多等好几天。发消息前，对照[物料清单](/zh/hardware/bill-of-materials/)再核一遍。
:::

## C 类——下单 3D 打印

这一步多数做软件的人没干过，所以换个角度想：你是把一组几何文件和一份材料规格交给打印服务，他们把成品零件寄给你。把这份订单写对，就是这步的全部工作。

### 要发什么

从 build123d 源文件把零件导出为 **STEP 文件**，打包成一个可下载的 zip 交给打印厂。一共有 **9 个唯一设计 / 11 件**：

- **惰轮支架 ×4**（2 个前置 + 2 个电机变体）
- XY 接头（左）
- XY 接头（右）
- 同步带夹
- solenoid 支架
- 手机座
- 控制板托
- 管夹

:::caution[忽略遗留的 FCStd 文件]
`hardware/custom/` 里有五个 `.FCStd` 文件。它们 **不是** 权威几何——别把这些发给打印厂。导出为 STEP 的 build123d 零件才是基准。
:::

### 材料和工艺——没有商量余地

照这个精确指定，别让打印厂以「便宜」为由把你劝去 FDM：

| 规格 | 取值 |
| --- | --- |
| 材料 | **黑色 PA12 尼龙** |
| 工艺 | **仅粉末床熔融（SLS 或 MJF）** |
| 密度 | 实心 |
| 层高 | 100 µm |
| 最小壁厚 | 1.0 mm |
| 表面处理 | 毛坯 |

:::caution[拒绝 FDM]
这些零件有侧向孔和悬垂。在 FDM 打印机上，**侧孔会变形**、**悬垂会塌**，滑块和 solenoid 就装不到位。SLS/MJF 在粉末床里成型、无需支撑，所以几何能做得很准。PA12 尼龙、SLS 或 MJF，没得商量。
:::

## A 类——下单现成件

B 和 C 一下完，就把 A 类里的一切都订上，让它们并行到货。有几样东西容易买少：

**紧固件——按包买，宁可多订。** 整套搭建用到 250 多件：M3 / M5 / M6 以及 1/4-20 的 BHCS / FHCS / SHCS，落入式和侧滑入式 T 型螺母、方螺母、10 个轴肩螺丝、12 个垫片、8 个隔套、6 个平角码、2 个电机角码、1 个转角码，还有 4 个缓冲脚垫。螺丝是按 20 / 50 / 100 一包卖的——按包买、留点备用，别去数精确个数。哪怕少一颗 M3，也不该把一个跨好几天的搭建给卡住。

:::tip[紧定螺丝松脱是反馈最多的故障]
买一支 **Loctite 243（乐泰 243 螺纹胶）**，涂在驱动带轮的紧定螺丝上。这是一份便宜的保险，防的正是大家搭完之后最常碰到的那个问题。
:::

**耗材——和其余件放在同一批订。** 这些和结构件分开发货，所以很容易忘了订，然后就卡住：

- 锂基润滑脂（**NLGI 2**）——导轨出厂带油；你要擦掉重新打脂
- Loctite 243
- 热缩管
- PTFE/FEP 管，**~400mm**——注意线是走在它 **外面**，不是穿过它
- 缠绕管（螺旋管）
- 0.3mm² 引线

**手边要备的工具。** 如果这些你还没有，一并加进订单：

- 2 / 3 / 5mm 内六角扳手
- 电动起子（建议——螺丝 *真的* 很多）
- 一把 **电烙铁 + 热风枪**——你要接线焊接；这不是免焊的搭建
- 一个用来对框架找方的平面
- 一个手机调音 App，后面给同步带调张力时用

## 推荐的下单顺序

交期是最长的那根杆子，所以要把它并行掉：

<Steps>

1. **重读手册、重核完整 BOM。** 切割长度和沉孔/攻丝标注必须第一次就对——切错或漏攻一处，就又是好几天的等待。

2. **先下 3D 打印单（C 类）。** 导出 STEP → 打 zip → 按 PA12 / SLS-或-MJF 规格上传给打印服务。这通常是交期最长的一项。

3. **再下铝型材切割单（B 类）。** 把切割/攻丝/沉孔规格发给供应商。

4. **订现成件（A 类）。** 步进电机、控制板、驱动、导轨、同步带、惰轮、solenoid、相机、电源、紧固件、耗材，以及任何你缺的工具——全放一批里订，好让它们在打印件和型材赶制时同步到货。

</Steps>
