title: 组装
description: 把整套装置搭起来——坐好手机、装上电容笔、把相机吊到正上方，再把线全接好。约一小时，不用焊接。
---

组装全是机械活：把手机坐进它的角里，把写字机器人的笔换成电容笔，把相机夹到正上方，再把所有东西接到一个带供电的 USB 集线器上。不用焊接，不用自制板——大约一小时，主要时间都耗在调相机角度上。

```text
        camera (gooseneck, ~25 cm up, looking straight down)
            │
            ▼
   ┌─────────────────────────┐
   │  pen-plotter arm        │
   │  X/Y gantry             │
   │  Z = solenoid + stylus  │
   │                         │
   │   ┌───┐                 │   ◄── phone seated in the corner,
   │   │ ▓ │  phone          │       TOP-LEFT registered to the
   │   └───┘                 │       holder corner
   └─────────────────────────┘
        │            │
     USB serial   12 V power      camera ─USB─┐
        │            │                        │
        └──── powered USB hub ── USB ──────────┘──→ your Mac
```

顺序有点讲究：先坐好手机，相机才有个实物可以对准；接线放最后，免得你摆放零件时被线拽动。

<Steps>

1. **坐好手机——先定角。**
   把防滑垫铺在写字机器人的平台上，摆好两个 L 形角块拼成一个角。把手机正面朝上放好，用力推进那个角里，让它的 **左上角顶住手机座的角**。就这一个细节，决定了标定能不能复用：手机每次都回到一模一样的位置，臂学到的屏幕坐标才一直有效。

   {/* PHOTO: 手机正面朝上坐进 L 形角块支架里，手机左上角与角块内侧的角贴平；下面能看到防滑垫。 */}

2. **把电容笔装进 Z 夹座。**
   取下写字机器人的笔，把电容笔滑进 Z 滑座的夹座里。调好它的高度：solenoid 断电时软笔头刚好停在玻璃 *上方* 一点，通电时则按到玻璃上。笔头要平着触屏，不能斜着碰。

   {/* PHOTO: 电容笔夹在 Z 滑座的夹座里，纤维笔头朝下对着手机屏幕，停在抬起（up）位时与玻璃留一道小缝。 */}

3. **把相机吊到正上方。**
   把鹅颈夹夹在桌面上，弯好它，让相机 **从正上方俯视屏幕中心，离屏约 25 cm**。取景要框住整块屏幕，留一点余量。「正上方」很要紧：agent 是从这张图里读取屏上各处的位置，视角一歪，它对东西在哪的判断就跟着歪。

   {/* PHOTO: 装在鹅颈上的 USB 相机正对着手机上方，镜头朝正下方，离屏大约 25 cm，整块手机屏幕都在画面里。 */}

4. **把所有东西接到带供电的 USB 集线器上。**
   把相机和控制板都插到 **带供电的 USB 3.0 集线器** 上，再把集线器接到你的 Mac。带供电的集线器能让板子和相机不至于把笔记本的 USB 口拉到掉压。如果你用有线连接，手机的截图桥接也走同一个集线器。

   {/* PHOTO: 带供电的 USB 集线器上插着相机的 USB 线和控制板的 USB-B 线，一根上行线接到 Mac。 */}

5. **给控制板供电。**
   给控制板接上 **两根线**：USB 接到集线器（数据、G-code），以及 **12 V** 圆头电源（电机和 solenoid 供电）。接上 12 V 后板子的 LED 应该亮起。先上 12 V，再上 USB，和 [固件](/zh/hardware/firmware/) 刷写时的上电顺序一致。

   {/* PHOTO: 控制板上两根线都接好了——USB-B 接集线器、12 V 圆头插上——电源 LED 亮着。 */}

</Steps>

:::caution[刚性就是一切——鹅颈被碰一下就得重新标定]
点击对不准，最常见的原因就是相机漂移。鹅颈发软、夹子被蹭动、手机重放时哪怕只偏一点点没顶满角，都会让屏幕从臂学过的那些位置底下挪开。把每个夹子都拧紧，鹅颈尽量短，任何一次磕碰都当成重新 [标定](/zh/set-up/calibrate/) 的理由。
:::
