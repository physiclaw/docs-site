PhysiClaw 硬件采用 [CoreXY](https://corexy.com/) 运动结构——两个电机协同驱动的龙门架，Z 轴用电磁铁驱动触控笔尖完成点击。整个组装大约需要 **两到三个小时**。

装配手册有完整清晰的步骤，可在线查看，也可下载 PDF；想了解整机装配可下载 3D 模型。动手前不妨先浏览硬件图集，对整机有个直观印象。

:::tip[没有 CAD 软件？]
3D 装配模型是 `.step` 文件，需要 3D 软件打开。推荐 [FreeCAD](https://www.freecad.org/)——免费、开源、跨平台。
:::

:::caution[装配注意事项]
以下几点装配手册里都讲过，但值得再强调一遍：

- **触控笔尖务必可靠接地**：可以接到已接地电脑机箱后侧的裸露金属处，不要接墙上插座，也不要接控制板的 GND。
- **调好摄像头位置**：打开系统自带的摄像头预览，确认手机屏幕在画面中平整、尽量占满，无倾斜或畸变。
- **A/B 电机同步带张力保持一致**：张力不一致会导致机械臂运动失准。
- **锁紧电机同步轮**：紧固螺钉上螺纹胶（蓝色可拆型）。电机不断正反转，反复换向易使螺钉松动，同步轮一旦打滑，运动就会出现漂移。
:::
