完整搭建约需 **$127** 的通用零件。这里没有任何定制件——机械臂是消费级笔式绘图机，
摄像头是普通 USB 网络摄像头。

| 部件 | 型号 | 约价 |
| ---- | ---- | ---- |
| GRBL 机械臂 | 派希笔式绘图机 P25（X/Y 龙门 + Z 舵机） | ~$80 |
| 顶部摄像头 | 绿联 1080P USB 摄像头，定焦 | ~$14 |
| 侧向摄像头 | 绿联 1080P USB 摄像头（同款） | ~$14 |
| 触控笔 | 电容触控笔，导电纤维笔尖 8–10 mm | ~$1.50 |
| 摄像头支架 | 金属鹅颈桌夹，50 cm（×2） | ~$4 |
| 手机支架 | 防滑垫 + L 形挡块 | ~$1.20 |
| USB 集线器 | 带电源的 USB 3.0 集线器 | ~$13 |

## 选件说明

- **机械臂**是唯一影响精度的部件。任何兼容 GRBL 的 X/Y/Z 绘图机都可以；
  P25 便宜、足够刚性、运行标准 G-code。
- **两个摄像头特意选同款**——只需熟悉一种固件脾气、一套驱动，可互换。
- **触控笔尖**应为柔软导电纤维，8–10 mm。硬橡胶尖触发不稳定，还会划伤屏幕。

## 组装概览

1. 将手机屏幕朝上放入绘图机工作台上的支架，用防滑垫和挡块固定。
2. 把绘图机的笔换成 Z 轴夹具中的电容触控笔。
3. 用鹅颈夹将**顶部摄像头**正上方垂直俯视屏幕。
4. 将**侧向摄像头**约 45° 夹好，使其能看到笔尖和玻璃表面。
5. 把所有设备接到带电源的 USB 集线器。

:::caution[刚性很重要]
鹅颈支架的晃动是漂移最常见的原因。夹紧支架、尽量缩短臂长，
任何碰动后都要重新[标定](/zh/start/quickstart/)。
:::
