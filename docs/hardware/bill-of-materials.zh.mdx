这是搭建篇的第一页。在你下单或开始打印任何东西之前先读它——它决定了你的预算和心理预期，本节后面的内容都依赖于此。

## 先读这一段：你要造的到底是什么

如果你是从 OpenClaw 过来的，以为只要给一台便宜的现成笔式绘图机装个触控笔就行，那现在就把这个印象清空。**PhysiClaw 是一台从零手工搭建的 CoreXY 桌面龙门。** 它不是套件，不是消费级绘图机，也没有哪个单一产品能买来「改装」一下就用。你要自己采购原始铝型材、按需切割攻丝、打印尼龙件，并亲手把整套运动系统装起来。

这台机器由三个子系统组成：

<CardGrid>
  <Card title="CoreXY 运动平台" icon="rocket">
    一套双电机 CoreXY 龙门，沿 X、Y 把触控笔送到手机屏幕各处。两个 NEMA 17 步进电机固定不动，驱动单条同步带回路；滑座骑在 MGN9H 直线导轨上。这是整套搭建的精度核心。
  </Card>
  <Card title="点触机构（Z）" icon="pencil">
    Z 轴是一个 12 V 推拉式 **电磁铁/螺线管**，既不是舵机，也不是绘图笔。它把电容触控笔啪地按下去完成点击，松开时抬起。干脆、可重复的敲击，正是让点击和长按读出不同效果的关键。
  </Card>
  <Card title="顶置相机" icon="laptop">
    一台 4K 自动对焦 USB 摄像头从正上方俯视屏幕。它负责读屏，并在点击前确认触控笔此刻落在哪里。没有第二台相机。
  </Card>
</CardGrid>

:::caution[花钱之前先管理好预期]
这是一个跨好几天的项目，不是一个下午能搞定的。其中有按需切割和攻丝、从打印和供应商处下单且交期要好几天的零件、一次固件刷写，以及一个磨人的标定/调试阶段。在下单任何东西**之前**，请通读整本搭建手册并核对完整 BOM——切割长度或零件数量上的小失误，返工代价很高。
:::

## 成本与工程量

### 成本

零件总成本约为 **$145 美元**。在中国采购约合 **¥1046**（按约 7.2 人民币/美元）。在美国/欧洲市场同样的零件会**更贵**——把下面的数字当作以中国为中心的下限，而不是固定价格。

这笔钱主要来自少数几个大项：

| 分组 | 约价（人民币） |
| --- | --- |
| 3D 打印尼龙件 | ¥190 |
| 相机（4K 自动对焦摄像头） | ¥200 |
| 直线导轨 + 滑座 | ¥120 |
| 控制板（MKS DLC32 V2.1） | ¥100 |
| 铝型材 | ¥60 |
| 步进电机（2× NEMA 17） | ¥50 |
| 电源（12 V 2 A） | ¥50 |
| TMC2209 驱动（×2） | ¥35 |

紧固件和小五金不是一个单独的大项：螺丝、T 型螺母、垫片、隔套都是按 **20/50/100 个一包** 卖的，所以即便单个很便宜，「小零件」加起来也不少。预算里要留出这一块。

### 工程量：四个阶段

<Steps>

1. **采购、打印、切割。** 买通用件，联系供应商做型材的**按需切割/攻丝/钻孔**，并把 STEP 文件上传到打印服务商打印尼龙件。供应链以中国为中心，交期以天计。见 [采购](/zh/hardware/sourcing/) 和 [打印件](/zh/hardware/printed-parts/)。

2. **组装。** 零件齐了之后，手册预估机械组装**约两到三小时**。见 [组装](/zh/hardware/assembly/)。

3. **刷写固件。** 通过 USB 把 FluidNC 加载到控制板上（浏览器里刷写）。这是软件设置之前必须先过的一道关。见 [固件](/zh/hardware/firmware/)。

4. **标定与调试。** 设定 CoreXY 同步带张力，给导轨重新上油并对齐，再调试电磁铁。这个阶段需要反复迭代、很磨人——提前做好准备。见 [调试](/zh/hardware/tuning/)。

</Steps>

## 零件清单

权威依据是 **13 节的搭建手册**，外加生成零件几何的 **build123d 工序模块**。本页是一份规划摘要——下单前请对照手册核对数量和尺寸。

### 框架——铝型材（按需切割/攻丝/钻孔）

所有型材都是黑色阳极氧化、欧标截面。长度和加工都很具体；把切割和攻丝交给供应商做（见 [采购](/zh/hardware/sourcing/)）。

| 龙门 X 横梁 | 1020，185 mm | 1 | — |

### 直线运动

| 零件 | 规格 | 数量 |

### 执行机构

| 零件 | 规格 | 数量 |
| --- | --- | --- |
| NEMA 17 步进电机 | 42 步，用于 X/Y | 2 |
| 电磁铁/螺线管（Z/点触） | 12 V 0.8 A 推拉式，10 mm 行程，0.2–6 N，2× M3 安装孔，孔距 15 mm | 1 |
| 电容笔头 | M3 内螺纹，导电头约 Ø7 mm | 1 |

笔头拧在电磁铁的**铁芯**上，替换掉电磁铁出厂自带的螺母。这就是 Z 机构——没有舵机，也没有笔。

### 紧固件与五金（约 250+ 个）

从零搭一台龙门需要大量小五金。预计**超过 250 个零件**，按 20/50/100 个一包卖（这也是为什么要单独留一笔「小零件」预算）：

- **螺丝：** BHCS / FHCS / SHCS，规格 M3、M5、M6，外加 1/4-20。
- **螺母：** 弹片式/滑入式 T 型螺母和方螺母。
- **五金：** 10 个轴肩螺丝、12 个垫片、8 个隔套。
- **支架：** 6 个平角支架、2 个电机支架、1 个角支架。
- **脚垫：** 4 个缓冲脚垫。

### 耗材

| 项目 | 备注 |
| --- | --- |
| 锂基脂 | NLGI 2，用于导轨 |
| Loctite 243 | 螺纹胶——见下方提醒 |
| 热缩管 | 用于接线 |
| PTFE/FEP 管 | 约 400 mm；**线走在管外** |
| 缠绕管 | 理线用 |
| 导线 | 0.3 mm² |

:::caution[紧定螺丝松动是反馈最多的头号故障]
给紧定螺丝涂 Loctite 243——尤其是驱动带轮上的紧定螺丝。紧定螺丝松动是搭建者反馈最多的单一问题。这里的螺纹胶不是可选项。
:::

### 工具

- 2 / 3 / 5 mm 内六角扳手
- 一把电动起子（推荐——螺丝很多）
- 用于接线的 **电烙铁 + 热风枪**——是的，这个搭建涉及焊接

## 3D 打印件

共有 **9 个独特设计 / 11 个件**。惰轮座要打印 **4 个**（2 个前置安装 + 2 个电机安装版本），再加上：XY 接头左、XY 接头右、同步带夹、电磁铁座、手机床、控制板支架、管夹。

:::caution[材料和工艺没有商量余地]
用 **黑色 PA12 尼龙、粉末床熔融工艺（SLS 或 MJF）** 打印。**拒绝 FDM**——侧孔会变形、悬垂会下塌，零件撑不住运动系统的精度。用提供的 STEP 文件交给打印服务商打印。几何来自 build123d 的 Python 零件，导出为 STEP 并打包成可下载的 zip——见 [打印件](/zh/hardware/printed-parts/)。
:::

## 真正要紧的是什么

不是每个零件都得做到好。把注意力花在真正重要的地方：

<CardGrid>
  <Card title="精度关键件" icon="warning">
    **直线运动**（导轨和滑座）、**同步带和带轮** 的质量，以及 Z 上一记 **干脆的电磁铁** 驱动。它们决定了点击是否落在助理瞄准的位置，以及长按和点击能否读出不同效果。买好零件，仔细上油对齐，认真调试。
  </Card>
  <Card title="可将就件" icon="approve">
    **相机**（任意 4K 自动对焦 USB 摄像头都行）和 **通用紧固件**。这些可以按价格挑。但别把这套逻辑套到整个搭建上——这是一台精度机器，不是即插即用的通用套件。
  </Card>
</CardGrid>
