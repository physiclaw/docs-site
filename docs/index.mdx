PhysiClaw gives an AI agent a physical finger: one overhead camera reads the screen and a
capacitive stylus on a CoreXY gantry taps it — no APIs, and no app on the phone. These docs go
from a parts list to a running agent, plus the tools, gestures, and built-in agent that drive it.

Be honest with yourself before you start: this is a hand-built machine. You source the parts,
3D-print the custom pieces, assemble a CoreXY gantry, flash firmware, then calibrate — a
multi-day project of roughly **$145+ in parts**, not an afternoon kit.
