PhysiClaw gives an AI agent a physical body to operate a phone: one overhead camera reads the
screen and a stylus on a 3-axis arm taps it — no APIs, and no app on the phone. These docs go
from a parts list to a running agent, plus the tools, gestures, and built-in agent that drive it.

New here? Start with the [Vision](/en/start/vision/) for why PhysiClaw exists, then the
[Introduction](/en/start/introduction/) for what it is and [how it works](/en/start/how-it-works/),
and [build one](/en/start/what-youll-build/).
