---

PhysiClaw gives an AI agent a physical body to operate a phone: a camera reads the screen and a
stylus on a 3-axis arm taps it — no APIs, and nothing installed on the phone. These docs go from a
parts list to a running agent, plus the tools and gestures it uses.

New here? Read the [Introduction](/en/start/introduction/), then [build one](/en/hardware/bill-of-materials/)
and [run it](/en/start/quickstart/).
