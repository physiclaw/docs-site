一切都已就绪。

启动服务：

```bash
physiclaw server
```

它会自动打开`硬件设置`页面，勾选 `自动模式`，点击`继续`，可自动完成硬件设置。


## 任务示例

确认 PhysiClaw 手机已经登录**微信**，并把常用应用都登录好——比如美团、京东、淘宝——同时设置好默认收货地址和支付方式。

用你自己的微信，和 PhysiClaw 手机上的微信互加好友，然后像聊天一样发消息告诉它做什么：

```text
帮我在美团上点一杯瑞幸美式。
明天天气怎么样？
明早 7 点把最新新闻发给我。
在京东七鲜买牛奶和鸡蛋。
帮我在 DeepSeek 上搜一下「xx 问题」。
```

:::tip[还没配置模型？]
智能体需要一个大模型提供商。如果还没配置，请见[获取 API Key](/zh/set-up/provider/)。
:::
