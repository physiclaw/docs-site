Everything's set up.

Start the server:

```bash
physiclaw
```

It opens the **Hardware setup** page in your browser, turn on **Auto mode** and click **Continue**. It runs every step on its own, stopping only when it needs you.

Or skip the browser and start auto mode right in the terminal:

```bash
physiclaw auto
```

## Example tasks

Make sure the PhysiClaw phone is signed in to **WhatsApp**, and that the apps you'll use are logged in too — DoorDash, Amazon, Instacart — with a default delivery address and payment method set.

From your own WhatsApp, add the PhysiClaw phone as a contact, then message it like a normal chat to tell it what to do:

```text
Order a latte on DoorDash.
What's the weather tomorrow?
Text me the top news at 7 a.m. tomorrow.
Buy milk and eggs on Instacart.
Look up "<your question>" on ChatGPT.
```

:::tip[No model set?]
The agent needs a model provider. If you haven't set one up, see [Get an API key](/en/set-up/provider/).
:::
