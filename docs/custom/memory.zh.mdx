title: 记忆
description: PhysiClaw 如何在多次会话之间记住东西——~/.physiclaw/memory 下的几个纯 Markdown 文件。
---

智能体被唤醒时，会加载 `~/.physiclaw/memory/` 目录下的 Markdown 文件，以维持跨会话的记忆连续性。

<FileTree>
- ~/.physiclaw/memory/
  - USER.md
  - memory.md
  - 2026-06-22.md
  - …
</FileTree>

| 文件 | 存放内容 | 维护者 |
| --- | --- | --- |
| `USER.md` | 你是谁、你的偏好、常用聊天软件 | 你 |
| `memory.md` | 会话中要求智能体记住的事项 | 智能体 |
| `YYYY-MM-DD.md` | 当天的活动日志，仅追加 | 智能体 |

## 示例

`USER.md` 是可选的，用来存放关于你的信息，省去每次重复告知智能体的麻烦。

```markdown
## 称呼
老张

## 聊天软件
微信（昵称：张三）

## 沟通
- 务必先回复消息，再执行操作
- 涉及付款，执行前先确认

## 常用信息
收货地址：北京市朝阳区 xx 路 xx 号

## 口味
不吃辣，咖啡要美式、不加糖
```

## 让它记住

你无需手动修改这些记忆文件。只要在对话中告诉 PhysiClaw 需要记住的内容，它就会自动写入 `memory.md`，往后的每次会话都会记得。

:::tip[发信息]
记住我不吃香菜，点外卖时记得备注。
:::
